## 术语

部分文档中会用到一些谷歌内部的术语，特在此说明:

*   **CL**: "changelist"的缩写，代表已经进入版本控制软件或者正在进行代码评审的变更。
    其他组织经常称为"change"或者"patch"。
*   **LGTM**: "Looks Good to Me."的缩写，当评审者通过代码评审时会这么说。
